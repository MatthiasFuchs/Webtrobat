Code template instructions
--------------------------
1. Install:
    1. Eclispe: Window/Preferences/JavaScript/Code Style/Code Template
    2. Click import: choose codetemplates.xml and apply changes

2. Using:
    1. Ctrl + N
    2. type in "javascript" and choose java script file
    3. go next, enter a name and finish it
    4. a new java script file with the specified name
       _and_ our Webtrobat header should be created
